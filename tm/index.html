A browser-based RPG set in the world of Ytecat. Built in HTML, one file at a time. This is not the finished product — but it is real, and it is playable.

⚠ Before You Play

This is an early development build. What you see here is not representative of the final game — it is a work in progress shared for curiosity and feedback. Features are incomplete, content is placeholder, and things will break. Relax, explore, and don’t @ me about the missing bits.

What’s Currently Playable
  • Character Creation— name, race, appearance, magic affinities and starting stats
  • Fenjiren— the first hub town, with locations to explore
  • Inventory— item management and character equipment screen
